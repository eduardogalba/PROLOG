      PROPERTYbasic_surface/1
      Defines a surface of charged cells represented by a list of lists. It is defined as: 
      basic_surface([[H|T]]) :-
    charge(H),
    my_list(T).
basic_surface([[H|T]|T2]) :-
    charge(H),
    my_list(T),
    basic_surface(T2).

      Note: basic_surface/1 must have at least one sublist with a charged cell.

      Usage:basic_surface(CellList)

      CellList is a list with charged cells.
